## ¿Qué es KNEKTED?
KNEKTED se plantea como una plataforma para **comercialización de información y material multimedia de valor** (p. ej. fotografías, reportes, documentos), orientada a eliminar intermediarios y mejorar la **capitalización del creador**.

## Problemas que ataca (resumen)
- **Centralización de la información**: grandes agencias/corporaciones capturan el valor.
- **Valoración**: el creador no tiene un “mercado abierto” que le ayude a descubrir el valor real.
- **Capitalización**: el proveedor suele ser el menos beneficiado.
- **Inmediatez**: el contenido pierde valor si no existe un canal global rápido.

## Propuesta de solución (alto nivel)
- Marketplace con reglas por **smart contracts**.
- Token de intercambio **KNT** (estándar ERC‑20 / Ethereum).
- Herramientas para publicar contenido: etiquetado, catalogación, watermark, precio, etc.
- Modelos de venta: **compra directa** y/o **subasta**.
- Mecanismo de reputación/calificaciones para vendedores.
